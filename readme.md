# Adaptive Cognitive Learning System

## Overview

This project is a research-oriented adaptive educational assessment platform designed for:
- M.Tech mini project
- seminar presentation
- technical paper publication

The system dynamically adapts assessment difficulty using:
- Thompson Sampling
- Bayesian proficiency estimation
- confidence analysis
- response-time analysis
- cognitive state detection

## Core Research Contribution

Unlike traditional assessment systems that rely only on correctness, this system additionally analyzes:
- confidence level
- hesitation patterns
- response time
- coding behavior

to personalize adaptive learning and assessment.
