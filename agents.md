# Development Instructions

This is a lightweight research-oriented FastAPI project.

Important Rules:
- Keep implementation simple and maintainable
- Avoid overengineering
- Use SQLite only
- Use FastAPI + SQLAlchemy
- Use explainable logic instead of heavy deep learning
- Prioritize adaptive intelligence and research evaluation
- Keep architecture beginner-friendly

Main Intelligence Components:
- Thompson Sampling
- Bayesian proficiency estimation
- Confidence analysis
- Hesitation analysis
- Cognitive state classification

Avoid:
- microservices
- distributed systems
- Kubernetes
- complex cloud architecture
- unnecessary abstractions

The project should remain suitable for:
- academic mini project
- seminar presentation
- technical paper publication
